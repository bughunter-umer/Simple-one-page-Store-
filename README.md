Ecommerce Store Dashboard

An interactive E-commerce Admin Dashboard built with HTML, TailwindCSS, and Chart.js.
It provides a clean, responsive layout for managing products, orders, customers, and sales reports — perfect for small businesses and demo projects.

🚀 Features

📊 Dashboard Overview

Displays total customers, orders, products, and revenue.

Interactive sales charts (line and bar) with Chart.js.

🛒 Products Management

Product listing with image, category, price, stock.

Options to Add, Edit, and Delete products (static demo).

📦 Orders Management

Order tracking with ID, customer, date, amount, and status.

Options to View or Cancel orders.

👥 Customers Section

Displays customer details (name, email, phone, total orders).

Options to View or Delete customers.

📑 Reports Section

Sales summary with total revenue, orders, and AOV (Average Order Value).

Top selling products list.

Monthly sales chart (bar chart).

🖥️ Responsive Sidebar Navigation

Switch between tabs (Dashboard, Products, Orders, Customers, Reports).

Active tab highlighting with smooth UI.

🛠️ Tech Stack

HTML5 – Structure of the app.

TailwindCSS – Modern utility-first CSS for styling.

Chart.js – For interactive charts and graphs.

Vanilla JavaScript – Handles tab navigation and chart rendering.
